## 📋 What's Inside

This is a **practical guide to the modern data analytics toolkit**—covering the five pillars every data analyst needs:

### **1. SQL — Querying & Structuring Data**
- `SELECT`, `WHERE`, `GROUP BY`, `HAVING`
- `JOINs` — Inner, Left, Right, Full
- Window Functions: `RANK()`, `ROW_NUMBER()`, `LAG/LEAD`
- CTEs & Subqueries for readable logic
- Query Performance & Indexing

**Applied in:** Amazon SQL Business Analysis (20+ queries)

### **2. Excel & MIS Reporting**
- Pivot Tables & Pivot Charts
- `VLOOKUP` / `XLOOKUP` / `INDEX-MATCH`
- Conditional Formatting & Data Validation
- Macros & Basic VBA
- Dashboard-style MIS Reports

**Applied in:** E-Commerce Sales Dashboard

### **3. Python for Data Analysis**
- **Pandas:** DataFrames, groupby, merge, pivot_table
- **NumPy:** Array operations & numerical computing
- **Matplotlib & Seaborn:** Static visualizations
- **Jupyter Notebook:** Iterative exploration

**Applied in:** Airbnb Bookings EDA (50,000+ rows)

### **4. Power BI & Data Visualization**
- Power Query — Cleaning & shaping data
- Data Modeling — Star schema & relationships
- DAX Measures — YoY growth, attrition rates
- Interactivity — Slicers, drill-through, bookmarks
- Storytelling — One dashboard, one message

**Applied in:** HR Attrition Dashboard (Power BI + DAX)

### **5. Statistics & EDA**
- Descriptive Statistics — Mean, median, mode, std dev
- Probability & Distributions — Normal curve, spread
- Correlation vs Causation — Rigor in analysis
- Hypothesis Testing — p-values, confidence intervals, A/B tests
- Exploratory Data Analysis workflow

## 📚 Key Concepts

### **Data Cleaning Best Practices**

| Issue | Fix |
|-------|-----|
| Missing values | Impute (mean/median) or flag & exclude deliberately |
| Duplicate records | De-duplicate using a unique business key |
| Inconsistent formats | Standardize dates, text casing & units |
| Outliers | Investigate the cause before capping or removing |
| Wrong data types | Explicitly cast columns (string, integer, datetime) |

### **Business Metrics by Domain**

**Sales & E-Commerce**
- Revenue, Average Order Value, Conversion Rate

**HR & Workforce**
- Attrition Rate, Headcount, Average Tenure

**Operations**
- Cycle Time, Utilization Rate, Cost per Unit

## 💡 Beyond the Tools: Communicating Insights

> **"A good analyst doesn't just find the answer — they make sure the right people understand it."**

### **Data Storytelling Principles**
1. **Lead with the insight**, not the chart
2. **Know your audience** — Executives want conclusions; analysts want methods
3. **Visual clarity** — Right chart type, minimal clutter
4. **Documentation** — Clear READMEs & case studies for reproducibility
